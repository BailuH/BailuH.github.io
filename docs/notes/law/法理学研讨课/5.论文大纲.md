# 以法律解释为核心的司法裁量在中国宪法语境下的方法论研究
## 选题
以法律解释为核心的司法裁量在中国宪法语境下的方法论研究

## 背景
司法裁量是连接抽象法律规则与具体个案裁判的关键枢纽，也是法理学与司法实践共同面临的核心命题。法律语言固有的模糊性、规范体系的结构性漏洞以及社会现实的极端复杂性，使得法官在裁判过程中不可避免地需要在多种合法备选方案之间作出选择。然而，裁量权的过度扩张又可能侵蚀立法权与司法权的分立边界，导致"法官假借法律解释之名，行立法之实"的宪政危机。
司法裁量作为一种“推定的裁量”，其本身具有很强的主观价值判断，因此如何划定司法裁量的边界与范围，不仅是一个认识论问题，更是一个**方法论**的问题。在“合宪性解释”时代，我们是否可以引入国家根本大法宪法作为法官在司法活动中的决断依据，并通过**法教义学**的理论与解释，为法官的司法裁量活动提供一个清晰的方法论呢？本文尝试研究一二。
