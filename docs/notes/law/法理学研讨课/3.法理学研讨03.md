# 普通法（系）法律人思维
## 亚当斯诉新泽西游轮案

这是一个==Hard Cases==（疑难案件），包含：
- new cases/novel cases，新案件/无先例可依的案件
- case first impression，第一次见的案件。

有两个==close cases==(相近判例):
- 旅店案
- 卧铺火车案

| 比较        | 相同点/相似点                                | 不同点                                 | 财产保障义务 |
| --------- | -------------------------------------- | ----------------------------------- | ------ |
| 游轮案与酒店案   | 1. ==封闭性空间== 2. 都提供住宿服务 3. ==客人的高度信赖== | 1. 静止与移动 2. 地上与水上 3.游轮除住宿服务外还提供运输服务 | 高      |
| 游轮案与火车卧铺案 | 1. 移动 2. 都是载具 3.都提供运输服务                | 1. ==封闭与开放== 2. ==信赖程度不同==          | 低      |
体现的是普通法思维的**归纳与类比**。
