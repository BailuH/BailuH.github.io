# 一、西周的法制
## 1.《九刑》与《吕刑》的区别
- 《九刑》是西周初期的立法，已有==“贼”、”藏“、”盗“、”奸“==等罪名，且不可赦免，维护了社会稳定性。
- 《吕刑》针对不同的犯罪行为将刑罚由重及轻分为==五刑、五罚、五过==；提出了审判官要==慎用刑罚==的原则，要区分犯罪者的主要动机，不能单凭口供；提出==疑罪从赎==的原则。

## 2.周礼——五礼的基本内容
- 吉礼、凶礼、宾礼、嘉礼、军礼(<u>嘉宾军吉凶</u>)
# 3.早期契约理论——”质“和“剂”
- 西周的买卖契约统称“质剂”
- 凡买卖奴隶、牛马等贵重的活物用较长的契约，称为“质”
- 凡买卖兵器、珍宝等贵重器物则用较短的契约，称为“剂”
## 4.审判制度——五听
- 案件审理中，法官”听狱之两辞“，凭借经验分析矛盾与真伪，称为”五听“，”听“是观察
- 五听包括：”辞听“（言辞是否矛盾）、”色听“（脸色是否异常）、”气听“（呼吸是否急促）、”耳听“（注意力是否集中）、”目听“（眼神是否游移）——<u>耳目辞听色气</u>
# 二、春秋战国（侧重成文法）
>法家的立法指导思想：——断于法——法布于众——轻罪重罚——刑无等级
## 1.《法经》
>战国时期，中国社会由奴隶制转向封建制（社会经济文化政治等方面分析），魏国首先颁布了第一部比较系统和完整的成文法《法经》，分为三个部分：第一部分为《盗法》《贼法》《囚法》《捕法》
- 为什么《法经》如此要重点打击盗、贼这两种犯罪？——“以为王者之政，莫急于盗贼，故其律始于《盗》《贼》”
- **历史意义**：作为中国历史上第一部==比较系统、完整的==封建法典，《法经》在中国法制史上具有重要的历史意义；首先，==《法经》使得魏国更加富强。==顺应了==历史发展==的潮流，使得==地主阶级政权==得到进一步巩固，==封建法制==得到进一步成长，==法律精神==得到进一步弘扬。这些都为魏国的进一步发展提供了动力，使其“强匡天下，威行四邻”；其次，《法经》是中国春秋时期==公布成文法的结晶==，取其精华，去其糟粕，代表了==春秋以来立法的最高水平==，在中国法制史上也是一个==里程碑==；最后，《法经》对以后的封建立法产生了很大的==影响==，以后的封建立法很多都建立在《法经》之上。

# 三、秦律
## 1.刑法原则——区分故意和过失
秦律称故意为“端”，过失为“不端”。

## 2.秦朝的刑种
秦朝法律的特征是实施严酷的刑罚。主要有五类刑罚。
1. 死刑
2. 肉刑、耐刑
3. 徒刑，徒刑指的是强制犯罪者劳作。包括城旦（男）、城舂（女）；鬼薪、白粲，男犯砍薪，女犯择米；隶臣妾，男为隶臣，女为隶妾；司寇、舂司寇。男犯伺察寇盗，女犯判司寇刑而劳作如舂；候。伺望敌情，在边地服役。
4. 流刑，
5. 财产刑
## 3.诉讼制度——“告”
- 秦代诉讼分为“告‘和”劾”。
- “告”指百姓举报他人犯罪
- “劾”指的是御史及其他官吏纠举犯罪
- 告发必须先经过佐官，不能直接到官长处告发。根据案件的性质以及涉案当事人的身份，秦律将“告”分为“公室告”（公共事务）和“非公室告”（家庭内务）。前者代表国家的官吏必须纠举，百姓无论作为被害人还是知见人，也得举告；后者家里人不得告发，即使告发官府也不得受理。
# 四、汉朝
## 1.汉朝的法律形式
- 律、令、科、比——到唐朝变为律、令、格、式
- 律是最基本的法律形式，是由中央朝廷制定并经皇帝批准颁布天下、长期适用、比较稳定的一种法律形式。
- 令也是汉代承袭秦制的一种法律形式，是由皇帝发布并经过整理纳入“令”系统的诏命。
- 科又称“科条”或‘事条“，是中央朝廷对律令条文的解释与细化。
- 比，又称”决事比“，是指在律令科条无明确规定的情况下，可以已经判决并经朝廷认定的典型案例作为司法审判的依据。
## 2.汉朝的徒刑体制（与秦律对比）
	汉朝的徒刑沿秦制，也称“作刑”。徒刑按照服刑期限可分为五种。
1. 髡钳城旦舂（五岁刑）：剃发戴铁钳，男犯筑城、女犯舂米，刑期五年。 
2. 完城旦舂（四岁刑）：保留头发，仅剔去鬓须，劳动内容同上，刑期四年。
3. 鬼薪白粲（三岁刑）：男犯为宗庙砍柴、女犯择米，刑期三年。
4. 司寇作（二岁刑）：男犯守边、女犯作杂役，刑期二年。
5. 罚作、复作（一岁至三月刑）：最轻的劳役刑，男犯戍边或杂作，女犯在官府服役，刑期一年至三个月不等。
## 3.汉文帝肉刑改革
- 汉文帝将一系列肉刑改为笞刑，但是笞刑过重，受刑者“率多死”
- 汉景帝的《箠令》，规定了用作笞刑刑具的“箠”的规格及行刑方式，但是又带来“死刑既重而生刑又轻”的刑法结构不合理的问题，这个问题直到南北朝才解决
- 汉文帝改革肉刑顺应了历史发展的趋势，虽未完全解决刑重问题，但为北朝及隋封建五刑的确立奠定了基础。
# 五、魏晋南北朝与隋唐
## 1.两晋的立法：《晋律》（《张杜律》、《泰始律》）
- 张斐、杜预
- ”准五服以制罪“是《晋律》首创，目的在“峻礼教之防”。它是指亲属间的犯罪，据五等丧服规定的亲等来定罪量刑。
## 2.隋法律内容
- 隋朝的刑事法律的“十恶重惩”原则：”十恶“中的“不道”指的是用一些巫术，或者特别残忍的方式进行杀人的方式
- 十恶具体有：==谋反、谋大逆、谋叛、恶逆、不道、大不敬、不孝、不睦、不义、内乱==（和唐一致）
## 3. 唐朝法律内容
- 司法机构：大理寺（审判）、刑部（司法行政）、御史台（检查）
- 唐朝的“十恶”：谋反、谋大逆、谋叛、恶逆、不道、大不敬、不孝、不睦、不义、内乱
- 唐朝的刑法原则：
1. 亲属犯罪增减处罚：人身伤害案件，尊长伤害卑幼，则亲属关系越近，处罚越减轻。反过来则相反；侵犯财产案件中，亲属关系越近，加害人减轻处罚越多
2. 举重明轻与举轻明重；
3. 外国人犯罪：同一国籍的外国人之间的案件发生在中国，适用该国法律；不同国籍的外国人，包括中国参与的犯罪，一律按唐朝法律适用；非法入境交易比照中国人非法出境交易治罪；外国人因出使进入中国境内从事货物交易活动，计赃准盗论
>唐朝的“数罪并罚”：
>基本原则是重罪吸收轻罪，判决书虽写明所有罪行，但是选择重罪处罚即可，体现了“宽仁”思想
>重与轻看的是刑罚的轻重而不是行为本身恶劣程度，但是例外是如果重罪可以“收赎”，轻罪不行的话，那么要选择轻罪。
>赃罪例外，原则是累计赃值，折半论处；如果是监守自盗或者一个事情多次受贿、同事共贿等加重情形就不折半论处
- 罪名体系中的“侵犯人身罪”分为“六杀”：故杀、谋杀、斗杀、戏杀、误杀、过失杀
- “侵犯财产罪”分为“六==**赃**==”：强盗、窃盗、受财枉法、受财不枉法、受所监临、坐赃
- 婚姻制：离婚的两种方式：协议离婚（和离），强制离婚（法定强制【违律为婚、法定“义绝”】、丈夫强制）
- 告诉的限制：1. 卑幼告尊长，奴婢告主人（不能以下犯上），但是如果是谋反、谋大逆、谋叛等重罪例外；在押囚犯和年到八十以上、十岁以下的无告诉权
- 不论是举劾还是告诉均要向当事人直接受管的府衙或主管部门提起，否则判“越诉”罪
# 六、宋朝法制（继承制）
## 1.司法机构
- ==大理寺==是中央最高审判机关，负责审理地方上报的刑事案件以及京城百官案件
- ==刑部==作为司法行政机关，负责复核大理寺审断的全国死刑等重大已决案件。审刑院的分出与归并
- ==御史台==是国家最高监察机关
## 2. 财产继承
1. 一般财产继承：第一顺序为儿子和女儿；第二为孙子和守寡妻妾；
2. 户绝继承：没有男丁，第一顺序为女儿（未嫁女继承全部，出嫁女继承三分之一，其余归官府）；第二顺序是近亲；第三顺序是官府
3. 遗嘱继承（前提是户绝）：缌麻（五服最远的）以上的亲属；遗嘱得经官府公证或族人见证；诉讼时效是十年
4. 中外客商死亡继承
- 有伴随者：若有父母妻子兄弟未嫁女等伴随，直接由他们继承；若无这些，由父母妻子持官府公文前来追认。
- 无伴随者：先由官府保管，并通知有继承权的人
## 3. 翻异别勘制度（要求再审）
- 当犯人不服判决临刑称冤，或者其家属代为伸冤时，改由另一个司法机关重申，或者由监司另派官员复审的制度
1. 换法官：移司别推
2. 换法院：差官别推、移案别推
- 翻异可三到五次，但实际上可以超过，体现的是“慎刑”的标准
# *元朝的司法机构（大变）
1. 大宗正府：负责审理蒙古和色目人（优越种族）
2. 刑部：审判和司法行政（吞并大理寺）
3. 特殊司法机关：
- 御史台：监察
- 枢密院：
- 宣政院：管理涉及僧侣的案件
# 七、明朝
## 1.司法机构
1. 刑部：负责审判和司法行政（依旧吞并大理寺职能）
2. 督察院：负责职官犯罪的司法审判机构
3. 大理寺：复核机关
## 2.《大明会典》
- 首次使用了“会典”这个名称
## 3.申明亭
- 张贴当地犯错的人名和事由，以示惩戒
- 里甲制度（调解制度）：每一里设“老人”一职，轻微案件必须先经过老人的裁决，不能直接告官

# 八、清朝
## 1.司法机构
**中央司法机构 - 三法司制度：**
- **刑部**：中央最高审判机关，职权范围远超明代，享有审判权、复审权与刑罚执行权，在三法司中权力最重
- **大理寺**：地位较明代下降，主要负责复核刑部拟判的死刑案件，行使慎刑职能
- **都察院**：法纪监督机关，审核死刑案件，参与秋审与热审，监督百官

**会审制度：**
- 死刑案件须由御史、大理寺官会同刑部司员复核，称"会小三法司"
- 再由都御史、大理寺卿与刑部尚书、侍郎复核，称"会大三法司"
- 三法司将审核结果奏报皇帝，由皇帝最终裁定
**地方司法机构：**
- 实行行政与司法合一，地方行政长官兼理司法
- 总督、巡抚、按察使、知府、知县等各级官员都负有司法职责
## 2.监察机构
**都察院：**
- 清代最高监察机关，被称为皇帝的"耳目风纪之司"
- 监察御史分道巡察，纠劾百官
- 参与司法监督，审核死刑案件，参加秋审、热审
**六科给事中：**
- 清初沿袭明制，六科独立办公
- 雍正元年（1723年）并入都察院，与监察御史合称"科道"
- 负责稽查六部政务、审核奏章、封驳诏敕
- 光绪三十二年（1906年）裁撤
**监察体系特点：**
- 科道官员品级虽低但权力很大，可直接向皇帝密奏
- 实行"风闻言事"制度，允许依据传闻弹劾官员
- 形成中央、省、地方三级监察网络
## 3.会审制度
### 1. 秋审：
- 每年秋季举行的死刑复核制度
- 由三法司会同九卿共同审理各省上报的死刑案件
- 将死刑犯分为"情实"、"缓决"、"可矜"、"留养承祀"等类别
- 是清代最重要的会审制度，体现慎刑思想
### 2. 朝审：
- 每年霜降后举行的京师死刑案件复核制度
- 由三法司会同公侯伯等贵族共同审理京师监候死囚
- 审理流程与秋审基本一致
- 体现对京师地区死刑案件的特别重视
### 3. 热审：
- 每年小满后至立秋前施行的特殊审判制度
- 主要审理轻罪案件，进行减等或释放
- 由大理寺官员与刑部官员共同审理京师笞杖案件
- 体现"暑热减刑"的仁政思想，疏通监狱
## 4. 《钦定宪法大纲》
**基本概况：**
- 1908年清政府颁布的中国历史上第一部宪法性文件
- 参照日本明治宪法和普鲁士宪法制定
- 确立二元制君主立宪政体
**主要内容：**
- **君上大权**：规定皇帝享有至高无上的权力，包括立法、行政、司法、军事等各方面大权
- **臣民权利义务**：首次以法律形式规定臣民的基本权利和义务，但置于附则地位
- **议院法要领**：规定议会的组织和职权，但议会权力有限
- **选举法要领**：规定议员选举制度
**历史评价：**
- 积极意义：首次以宪法形式规定臣民权利，开启民智，培养近代法律意识
- 局限性：实质是维护君主专制，君权宪法化，民权地位低下
- 被批评为"吐饰耳目，敷衍门面"，激起立宪派不满
## 5. 礼法之争
**历史背景：**
- 清末修律过程中发生的重大理论争执
- 以沈家本为代表的"法理派"与张之洞、劳乃宣为代表的"礼教派"之间的争论
- 实质是旧法与新法、传统与现代的冲突
**主要分歧：**
**礼教派观点：**
- 主张=="应本旧律之义，用新律之体"==
- 强调将传统礼教内容纳入新形式法律
- 维护三纲五常等封建礼教传统
- 认为新律破坏中国名教，有违"君为臣纲、父为子纲、夫为妻纲"
**法理派观点：**
- 主张用西方资产阶级法律原则改良传统法
- 强调"折衷各国大同之良规，兼采近世最新之学说"
- 认为法律应体现自由、民主、平等和法治精神
- 主张法律与礼教分离
**争论结果：**
- 清政府基本采纳礼教派意见
- 《刑事民事诉讼法》被废弃，重新制定独立的诉讼法
- 《大清新刑律》被迫修改，增加维护礼教条款
- 反映清末法律改革的保守性和局限性
## 6. 司法机构改革
**改革背景：**
- 清末新政和预备立宪的推动
- 西方司法制度的影响
- 传统司法制度的弊端日益显现

**主要改革措施：**
1. **司法行政分离：**
- 1906年进行官制改革，开始实行司法与行政分离
- 改刑部为法部，专掌司法行政
- 改大理寺为大理院，专掌审判
2. **建立新式审判机构：**
- 设立各级审判厅，建立四级三审制
- 推行法官考试制度，培养专业司法人才
- 建立律师制度，实行辩护制度 
3. **改良监狱制度：**
- 改革传统监狱管理制度
- 推行感化教育，改善囚犯待遇
- 建立新式监狱，采用现代管理理念

**改革特点与局限：**
- 改革具有渐进性和保守性
- 受传统势力阻挠，改革不彻底
- 受经费和人才限制，实际效果有限
- 为民国时期的司法制度奠定基础
