## 一、 英国：宪政的先行者

- **起源**：源于封建契约制度 。
- **《大宪章》**：确立了“议会至上”与“王权有限”的原则 。
- **光荣革命**：封建贵族与资产阶级的非暴力夺权 。
- **《权利法案》**：正式确立宪政原则，建立虚君共和，保障议会辩论自由 。
## 二、 美国：有限政府与联邦制

- **核心观念**：政府不得越权（有限政府），权力来源于被统治者的同意 。
- **制宪成果**：通过各州妥协产生联邦宪法，确立参众两院制 。
- **结构特点**：正文侧重于通过制约政府权力来保障个人自由 。
- **关键机制**：利用联邦制和多元主义削弱“多数人暴政”，保障少数派系自由 。
- **演进时刻**：建国（人权起步）、内战（正当程序与平等保护）、新政（强化联邦调控） 。
## 三、 法国：集权与分权的实验

- **早期发展**：1789年《人权宣言》奠定基本原则；早期长期拒绝司法审查，偏向行政集权 。
- **第三、四共和**：议会至上，但立法与行政权平衡失调 。
- **第五共和（现行）**：强化总统权，建立宪法委员会进行合宪性审查 。
## 四、 德国：集中审查模式

- **历史曲折**：1848年宪法未生效，1871年宪法带有专制色彩，1919年《魏玛宪法》民主基础薄弱 。
- **1949年《基本法》**：确立专门法院系统，创立了对世界影响深远的“集中审查模式” 。
## 五、 日本：从天皇专权到象征制

- **1889年宪法**：确认天皇总揽统治权，带有军国主义隐患 。
- **1946年宪法**：受麦克阿瑟影响，天皇转为国家象征，实行议会内阁制，宣布放弃战争 。
## 六、 社会主义宪法与全球趋势

- **社会主义模式**：始于1918年苏俄宪法，核心为生产资料公有制、人民民主专政及党的领导 。
- **发展趋势**：强调人权保障、行政权力扩大化（需限权）、重视违宪审查、向国际法扩展 。
