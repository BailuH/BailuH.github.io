## 一、 清末预备立宪（1895-1911）

- **动机**：带有强烈的实用主义色彩，意在强国 。
- **《钦定宪法大纲》**：实权君主制，强调皇帝权力至高无上，是维护专制的工具 。
- **代议制萌芽**：设立资政院（中央）与省咨议局（地方） 。
- **《十九信条》**：辛亥革命压力下的妥协，接近英国虚君立宪制 。

## 二、 中华民国时期（1912-1948）

- **《临时约法》（1912）**：中国首部近代意义及资产阶级性质的宪法，确立主权在民与责任内阁制 。
- **袁世凯时期**：1914年《民国约法》（天坛宪草）确立个人独裁，摧毁国会 。
- **北洋与联省自治**：1923年“贿选宪法”本质上属于联邦宪法，确认分权制度 。
- **国民政府时期**：实行“党治”；五权宪法思想（三权+监察+考试）；1946年宪法最终形成党治宪法 。

## 三、 中华人民共和国时期（1949-至今）

- **早期文献**：革命根据地的宪法大纲；1949年《共同纲领》起临时宪法作用 。
- **1954年宪法**：第一部社会主义类型宪法，规定了国体、政体及总任务 。
- **演变与拨乱反正**：历经75宪法、78宪法，最终制定现行**82宪法** 。
- **82宪法特点**：权利保护移至国家机构前；规范性加强；完善人大机制 。

## 四、 1982年宪法的五次重大修订

1. **1988年**：私营经济地位、土地使用权可转让 。
2. **1993年**：确立社会主义初级阶段、市场经济、家庭联产承包责任制 。
3. **1999年**：写入邓小平理论、依法治国、分配制度现代化 。
4. **2004年**：增加“国家尊重和保障人权” 。
5. **2018年**：设立宪法和法律委员会、监察委员会；赋予设区的市地方立法权 。
