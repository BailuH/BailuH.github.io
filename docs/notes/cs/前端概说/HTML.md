#### 特殊文本符号
在HTML中，特殊文本符号通常被称为**字符实体**（Character Entities）或**HTML实体**。
由于HTML解析器会将某些字符（如 `<` 和 `>`）视为代码的一部分，因此如果要直接在页面上显示这些字符，或者显示键盘上无法直接输入的符号（如版权符号 ©、欧元符号 €），就需要使用实体。
实体主要有两种写法：
1. **命名实体**：例如 `&nbsp;`
2. **数字实体**：例如 `&#160;` (十进制) 或 `&#xA0;` (十六进制)

这个不用自己去背，需要时查询即可：
[随手找的一个在线查询网站](https://www.iicn.net/tools/html-entities.html)

### 图像
插入图像标记`<img>`，拥有以下常用属性：
1. `src`：图片的的URL路径，可以是==本地资源的相对路径，也可以是网络资源==。
2. `width`：图像的宽度
3. `height`：图像的高度
4. `alt`：定义可以替换页面图像的文本。
还有其他的更详细的信息参见：[MDN的HTML指南](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/img)

### 表单/文本框
标记`<input>`,拥有以下属性：
1. `type`：最重要的属性之一，用来指定表单的类型。
> 💡关于`HTMLInputElement`
> 由于每一个`<input>`元素(element)，**不论是何种类型**，都是基于`HTMLInputElement`接口。
> `HTMLInputElement`是**Web API**的一个接口，可以理解成浏览器内部对`<input>`标记的编程定义。
> 🤔`<input>`的属性特点
> 从技术实现看，所有的`<input>`共享同一套属性，因此从JavaScript接口的角度来看，它们拥有完全相同的**属性**集合；
> 但从实际应用来看，并不是所有属性对于特定的`type`都会起作用，因此这些属性总是“分组”的。

| 表单类型      | 可选的type值                       |
| --------- | ------------------------------ |
| *文本(默认值)* | `text`                         |
| 密文        | `password`                     |
| 按钮        | `button`                       |
| 复选框       | `checkbox`                     |
| 单选框       | `radio`                        |
| 颜色选择器     | `color`                        |
| 日期选择器     | `date`/`datetime-local`(带时间)   |
| 电子邮件输入    | `email`                        |
| 文件选择      | `file`(配合`accept`属性指定可接受的文件类型) |
| ...       |                                |
由于属性总是==“分组”==的，因此最好的方法是按照功能来分组，并记住分组的属性与用法。
