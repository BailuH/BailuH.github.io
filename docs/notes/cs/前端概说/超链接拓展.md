```html
  <!-- 4. javascript: 执行 JS -->
  <h2>⚡ 4. javascript: - 执行 JavaScript 代码</h2>
  <a href="javascript:alert('Hello from javascript: URL!')">点击弹出警告框</a>
  <br>
  <a href="javascript:void(0)" onclick="console.log('Clicked via onclick')">配合 onclick 使用（推荐方式）</a>
  <div class="note">
    ⚠️ 不推荐直接使用 <code>javascript:</code>，因为：
    <ul>
      <li>违反内容与行为分离原则</li>
      <li>可能被安全策略阻止（CSP）</li>
      <li>不利于 SEO 和无障碍访问</li>
    </ul>
    更好的做法是使用 <code>onclick</code> 事件监听器。
  </div>
```
