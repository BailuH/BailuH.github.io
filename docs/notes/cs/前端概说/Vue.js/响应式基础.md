## 使用`ref`
在`Composition API` 中，推荐的声明响应式状态的方法是使用 `ref()` 函数：
```js
import { ref } from 'vue'

const count = ref(0)
```
`ref()` 接收参数并将其包装在一个带有 `.value` 属性的 ref 对象中返回：
```js
const count = ref(0)

console.log(count) // { value: 0 }
console.log(count.value) // 0

count.value++
console.log(count.value) // 1
```

实践中往往常见的写法组合就是*SFCs（单文件组件）+`script setup`的方式*进行开发：
```vue
<script setup>
import { ref } from 'vue'

const count = ref(0)

function increment() {
  count.value++
}
</script>

<template>
  <button @click="increment">
    {{ count }}
  </button>
</template>
```
