## 常见技巧
### 1.如果相关系数$R^2$很低
原因可能是：
1. 因为自变量本身对因变量的影响并不大，考虑增加自变量个数（一元to多元或者多元变更多元）
2. 可能是脏数据问题，进行==残差分析==。
3. 也有可能现有的数据表现的特征并不是线性相关，需要进一步做数据变换
### 2. 模型很复杂
如果建立的模型变量特别多，形式很复杂，很有可能是因为目前的==模型假设太过简单==，或者是==自变量选择太过随意==，导致引入了不相干的扰动因素，或者忽视了某些重要机理。
常见的我们考虑利用；
1. ==主成分分析==、==逐步回归==等方法确定几个最重要的变量，再建立模型
2. 模型可能有明显的==分段、周期==特征，应当分类讨论

### 3. 残差分析
残差分析是用来检验线性回归模型的重要手段，通过观察残差（$e_i​=y_i​−\hat{y}_i​$ ）的分布模式，可以系统性地发现模型缺陷和数据问题。
1. 验证模型基本假设（Gauss-Markov假设）

| 检验目标     | 残差表现                             | 若违反的后果         | 解决方向                      |
| -------- | -------------------------------- | -------------- | ------------------------- |
| **线性关系** | 残差 vs 拟合值图呈现**曲线模式**（如U型、倒U型）    | 参数估计有偏         | 添加多项式项、对数/平方根变换           |
| **同方差性** | 残差散布呈**喇叭形**（ funnel shape）或扇形扩散 | 标准误估计错误，假设检验失效 | 加权最小二乘(WLS)、对数变换、稳健标准误    |
| **正态性**  | Q-Q图上残差点**明显偏离对角线**（特别是尾部）       | 小样本下推断失效       | Box-Cox变换、非参数方法、Bootstrap |
| **独立性**  | 残差 vs 时间/顺序图显示**周期性或趋势性**        | 标准误严重低估        | 添加滞后项、广义最小二乘(GLS)、聚类标准误   |
2. 发现模型建立的不合适之处

| 残差模式        | 暗示问题             | 诊断方法               | 改进策略                          |
| ----------- | ---------------- | ------------------ | ----------------------------- |
| **分组/分层模式** | 遗漏分类变量或交互项       | 残差按某变量分组着色后呈现不同水平线 | 添加虚拟变量、交互项 ($X_1 \times X_2$) |
| **系统性偏移**   | 遗漏重要连续变量         | 残差 vs 未纳入变量图显示趋势   | 将该变量加入模型                      |
| **异质性方差**   | 不同子群体方差不同        | 残差 vs 某分类变量散布宽度不一致 | 分层建模、异方差稳健估计                  |
| **边界聚集**    | 因变量有截断（如预测值出现负值） | 残差在边界处堆积           | Tobit模型、对数变换、GLM              |
3. 识别问题数据点（数据质量控制）

| 类型        | 定义           | 残差指标                      | 处理方式             |
| --------- | ------------ | ------------------------- | ---------------- |
| **离群值**   | $y$值异常但$X$正常 | 标准化残差 $\|r_i\| > 3$       | 检查录入错误，稳健回归，或剔除  |
| **高杠杆点**  | $X$值极端（远离质心） | Hat值 $h_{ii} > 2(p+1)/n$  | 检查采样偏差，缩小影响（WLS） |
| **强影响点**  | 同时离群+高杠杆     | Cook距离 $D_i > 4/n$ 或 $>1$ | 必须检查是否为真实数据，考虑剔除 |
| **拟合不佳点** | 模型对其预测极差     | 学生化删除残差过大                 | 检查是否为特殊子群体，需单独建模 |
4. 检测特殊数据结构

| 现象       | 残差特征                    | 含义             | 应对                         |
| -------- | ----------------------- | -------------- | -------------------------- |
| **自相关**  | 残差 vs 时间/索引图呈**锯齿状或趋势** | 时间序列相关性或空间相关性  | 添加AR项、Newey-West标准误、混合效应模型 |
| **异常组**  | 残差图中出现**密集簇+孤立点**       | 数据来自不同总体（混合分布） | 潜类别分析、分样本回归、聚类后建模          |
| **测量误差** | 残差方差随$X$增大而增大           | 因变量测量精度不均      | 加权回归、工具变量法                 |

### 4. 与时间序列相关
时间序列中变量的顺序观测值之间存在**自相关**，因此我们需要需要诊断并消除自相关性，并建立一个自回归模型。
