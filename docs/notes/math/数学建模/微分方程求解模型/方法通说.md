用人话讲就是三个考虑要素：==是否有已知的科学规律？==、==是否可以进行微元分析？==、==是否可以提出某种合理的假设，在该假设下的数据拟合效果怎么样？==

## 📌 三种列微分方程的方法回顾：
### （1）按规律直接列方程
→ **已知物理/化学/生物等基本定律，可直接写出微分方程。**
✅ 特点：有明确公式或守恒律可用，如牛顿第二定律、Fick扩散定律、Malthus人口增长等。
### （2）微元分析法与任意区域上取积分的方法
→ **从局部微元出发，建立平衡关系，再取极限得微分方程；或对整体区域积分后求导得到微分方程。**
✅ 特点：适用于连续介质问题（如热传导、流体力学），需要从“小段”推导“整体”。
### （3）模拟近似法
→ **当机理不清时，基于数据或经验提出假设，构造一个能拟合现象的微分方程模型。**
✅ 特点：黑箱或灰箱建模，依赖数据和合理性假设，常用于生物、经济、社会系统。
