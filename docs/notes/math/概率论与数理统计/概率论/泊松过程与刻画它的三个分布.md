我当时的思维链大概是这样的：
1. 泊松分布刻画的是单位时间内某个随机事件以速率 $\lambda$ 发生的计数，仅具有一个参数
2. 那当发生的次数与时间长度 $t$ 有关，具体地说是泊松分布的参数从 $\lambda$ 变成 $\lambda t$ 之后，是不是说明这个泊松分布是一个随==时间变化==的动态分布？
3. 但是如果我带着这样的思路去考虑泊松分布与指数分布的关系的时候，越来越觉得这个参数 $t$ 十分诡异。我们通常==只会用一个参数==去刻画泊松分布，比方说一个上午客服接电话的次数，我们总是会先规定好“**单位时间**”是什么，如果我只知道 $\lambda t$ 的取值，我要怎么确定 $\lambda$ , 进而确定指数分布的参数呢？毕竟从目前的状况来看，不论时间 $t$ 取什么值，对应的故障间隔服从的指数分布的参数都只是 $\lambda$ ，不论是单位时间（$t=1$)还是t取到无穷大。
4. 对啊！单位时间和无穷大都一样，泊松分布不就是服从==可加性==的吗？那么这么一个看的很诡异的“泊松分布” $P(\lambda t)$ 不就可以被分解为 $t$ 个标准的泊松分布 $P(\lambda)$ 了吗？

>后来查了一下资料，像泊松分布这样具有可加性的随机变量，只要在时间长度 $t$ 上进行一个“叠加”，就能构造出**随机过程领域**中的=="独立增量过程"（Lévy 过程）==了。
>于是，通过泊松分布，我们可以构造出==泊松过程==——

## 泊松过程
### 两种等价的定义
**定义A：增量视角（茆诗松教材常用）** 设 N(t) 为时间 $[0,t]$ 内事件发生次数，满足：
1. $N(0)=0$ （从零开始计数）
2. **独立增量**：不相交时段内发生次数互不影响
3. **平稳增量**：任意长度为 t 的时段内，事件发生次数 $N(t)∼Poisson(\lambda t)$
4. **稀有性**：在极短时段 $\Delta t$ 内，事件发生概率很小（$P(\text{发生})≈ \Delta \lambda t$ ）

**定义B：等待时间视角（更直观）** 事件首次发生等待时间  $T_1$​ ，之后各次间隔时间 $T_2​,T_3​,…$ 满足： $T_1​,T_2​,T_3​,… \stackrel{\text{i.i.d.}}{\sim} \text{Exp}(\lambda)$ ，即：**泊松过程 = 独立同分布的指数分布等待时间串联起来的计数器**。

### 一些关键的性质

| 性质        | 内容                                                                                            |
| --------- | --------------------------------------------------------------------------------------------- |
| **计数分布**  | $N(t)\sim \text{Poisson}(\lambda t)$，$E[N(t)]=\lambda t$，$\text{Var}[N(t)]=\lambda t$         |
| **等待时间**  | 第 $n$ 次发生时刻 $S_n = T_1+\dots+T_n \sim \text{Gamma}(n,\lambda)$（即 Erlang 分布）                   |
| **无记忆性**  | 已等待多久不影响未来等待时间的分布（继承自指数分布）                                                                    |
| **分解/合并** | 速率为 $\lambda$ 的泊松过程可按概率 $p$ 拆分为两个独立的泊松过程，速率分别为 $\lambda p$ 和 $\lambda(1-p)$；反之，独立泊松过程相加仍为泊松过程 |

### 与分布家族的联系
```text
泊松过程 N(t)
    │
    ├── 固定时刻 t 的"计数" ──→ N(t) ~ Poisson(λt)  （离散分布）
    │
    ├── 相邻事件的"等待间隔" ──→ Exp(λ)  （连续分布）
    │
    └── 第 n 次事件的"等待总时间" ──→ Gamma(n, λ) / Erlang(n, λ)
```
