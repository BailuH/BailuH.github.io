## 聚类分析概述

**核心思想**：无需预先定义分类标准，直接==从样本数据出发，按亲疏程度自动分类==。同类事物具有高度相似性，异类事物具有较大差异性。

**作用**：
- 自动辨认具有相似性的事物
- 揭示同一类别中不同变量或样本的亲疏程度
- 通过变量/样本的连接状况辅助解释分类结构

## 距离和相似系数

### 变量测量尺度的类型

| 尺度类型 | 特征 | 示例 |
|---------|------|------|
| **间隔尺度** | 用数量表示，可为连续型或离散型；存在绝对零点时又称**比例尺度** | 长度、重量、收入、支出 |
| **顺序尺度** | 无明确数量，仅有次序关系（序数效应），相邻数值间无绝对基数效应 | 优、良、中、差 |
| **名义尺度** | 既无数量表示，也无次序关系 | 性别（男/女）、四季 |

### 样品间亲疏程度的测度

**两类数量指标**：

| 指标 | 用途 | 判定标准 |
|-----|------|---------|
| **距离** | 样品聚类（Q型） | 距离越近越亲密，归为一类 |
| **相似系数** | 变量聚类（R型） | 绝对值越接近1越密切，越接近0越疏远 |
> 在实践中，聚类的对象可以分为两类——==样本聚类==和==变量聚类==（横向和纵向聚类，想象数据是一个二维表），一般来说样本聚类使用**距离**作为指标，变量聚类使用**相似系数**作为指标。
#### 距离的定义
以 $d_{ij}$ 表示第 $i$ 个样本与第 $j$ 个样本间的距离，需满足非负性、同一性、对称性、三角不等式四个条件。
- $d_{ij} \geq 0$
- $d_{ij} = 0,\text{当且仅当i=j时成立}$
- $d_{ij} = d_{ji}$
- $d_{ij} \leq d_{ik}+d{kj}$

#### 常用距离的分类

| 距离名称                           | 公式特征                          |
| ------------------------------ | ----------------------------- |
| **欧氏距离** (Euclidean)           | 各变量差值平方和再开方                   |
| **欧氏距离平方** (Squared Euclidean) | 各变量差值平方和（不开方）                 |
| **切比雪夫距离** (Chebychev)         | 各变量差值绝对值的最大值                  |
| **马氏距离** (广义欧式距离)              | 各变量差值绝对值的总和                   |
| **明可斯基距离** (Minkowski)         | 各变量差值绝对值的 $p$ 次方总和，再求 $p$ 次方根 |
| **自定义距离** (Customized)         | 各变量差值绝对值的 $p$ 次方总和，再求 $q$ 次方根 |

> **自定义距离的特例**：$p=2,q=2$ 为欧氏距离；$p=2,q=1$ 为欧氏距离平方；$p=1,q=1$ 为马氏距离。

### 相似系数的计算

| 系数                | 定义与适用场景                           |
| ----------------- | --------------------------------- |
| **夹角余弦**          | 从向量集合角度定义，适用于**形状相似而长度不是主要矛盾**的场景 |
| **相关系数** $c_{ij}$ | 即常用的 $r_{ij}$，反映**线性相关**程度        |

### 相似系数和距离的转化
有的时候根据需要相似系数与距离可以相互转化，但要注意转换公式并非数理推导，而是一种==保持统计意义的拟制==。
- 若 $d_{ij}$ 为距离，可转化为相似系数：如 $c_{ij} = \frac{1}{1+d_{ij}}$
- 若 $c_{ij}$ 为相似系数，可转化为距离：$d_{ij} = 1-c_{ij}$ 或 $d_{ij} = \sqrt{1-c_{ij}^2}$

## 类间距离计算方法

>一旦我们将若干样本给划分为“某一类”时，这个时候如何去定义“类”与样本点的距离、”类“与”类“的距离就值得考虑。事实上，根据==系统（层次）聚类分析==的思想，所有的样本点其实在最初状态都应看作”类“，按照一个统一的方法进行”一视同仁“地聚类（而不管类中有一个或多个样本点）

设 $D(p,q)$ 为类 $p$ 与类 $q$ 之间的距离：

| 方法                                   | 定义                   | 性质/备注                   |
| ------------------------------------ | -------------------- | ----------------------- |
| **最短距离法** (Nearest Neighbor)         | 两类所有样本对距离的最小值        | 实际工作者常用                 |
| **最长距离法** (Furthest Neighbor)        | 两类所有样本对距离的最大值        | 不具有最优化性                 |
| **类间平均距离法** (Between-groups Linkage) | 两类所有样本对的平均距离         | 充分利用两类中所有样本信息（默认推荐）     |
| **类内平均距离法** (Within-groups Linkage)  | 两类所有样本对（含类内样本对）的平均距离 | —                       |
| **重心聚类法** (Centroid)                 | 两类重心（各变量均值点）间的距离     | 不具有单调性                  |
| **离差平方和法** (Ward's Method)           | 使类内欧氏距离总平方和增加最小的两类合并 | 分类效果好，**要求样本距离必须为欧氏距离** |
| **中位数法** (Median)                    | 两类中位数间的距离            | 比较稳健，不具有单调性             |
💡注一些相关性质的解释：
- 最优化性
	- **本质**：每次合并两类时，是否遵循某个明确的"目标函数最优准则"。
	- **Ward法（离差平方和法）具有最优性**：它每一步都选择"使类内欧氏距离总平方和增量最小"的两类合并，等价于在局部严格求解一个最小化目标，聚类结构紧凑。
	- **最长距离法不具有最优性**：它只关注两类中最远的一对样本，容易因个别异常点产生"链式效应"，把本不该归为一类的样本强行串连，无法保证全局结构最优，也没有明确的目标函数支撑。
- 单调性
	- **本质**：合并距离是否随聚类步骤严格递增，不出现"倒挂"。
	- **具有单调性**：若第 $k$ 步合并的类间距离记为 $d_k$，则满足 $d_1 \leq d_2 \leq \dots \leq d_{n-1}$。树状图（谱系图）从左到右平滑展开，不会交叉，聚类顺序一目了然。最短距离法、最长距离法、类平均法、离差平方和法均满足。
	- **不具有单调性（重心法、中位数法）**：两类合并产生新重心/中位数后，下一步与其他类计算距离时，可能出现**新距离反而比上一步更小**的情况，导致树状图出现"反转"（inversion），聚类层级交叉，难以直观解释。

-  稳健程度
	- **本质**：方法对异常值（outliers）和数据扰动的抗干扰能力。
	- **高稳健（中位数法）**：以中位数作为类代表点，极端值几乎不影响中心位置，结果稳定。
	- **低稳健**：
	  - **重心法**：使用均值作为类中心，单个极端值就能把重心拉偏；
	  - **最短距离法**：只看两类中最近的一对样本，一个异常点就可能拉近两个本不该合并的类；
	  - **最长距离法**：受最远点支配，同样容易被异常值扭曲。

## 层次聚类分析法（系统聚类分析）

### 基本思想
将 $n$ 个样本视为 $n$ 个独立类，每次合并最亲密的两类，经 $n-1$ 次合并后所有样本聚成一大类。

### 具体步骤
1. **选择变量**：根据研究问题确定分析变量
2. **数据标准化**：消除量纲或数量级差异，使结果更具科学性
3. **计算亲疏程度**：选择距离/相似系数公式，计算所有样本两两距离，得到距离矩阵 $D = (d_{ij})_{n \times n}$
4. **选择聚类方法**：合并最亲密的两类
5. **迭代合并**：若类的个数 > 1，重复步骤3-4
6. **输出图形**：聚类结果、树形图（谱系图）、冰柱图
7. **决定类数**：根据研究背景确定最终类的个数与含义
8. **科学解释**：结合领域知识得出分类结论

其中==由研究者决定==的比较重要的是*选择变量、选择聚类标准（何种距离or相似系数）、决定类数*。其中**决定类数**是最重要的，需要**综合样本特征**（主要考察样本==聚类优化过程==中==类间距离的变化==）、研究问题的**实际意义**等角度去考量。


## 14.5 快速聚类分析（K-Means Cluster）

### 概述
- 也称**动态聚类**，方法简单、占用内存少，适合**大样本**
- 实质是**分步聚类**：先选定初始类中心 → 样本向最近中心靠拢 → 调整中心 → 迭代至合理

### 具体步骤
1. 选择分析变量，指定聚类数目 $K$（$2 \leq K \leq$ 样本数）
2. **确定初始类中心**：
   - 用户自行指定 $K$ 组数据
   - 或系统根据数据自动选择 $K$ 个代表性样本
3. **计算欧氏距离**，按就近原则分派样本，完成一次迭代，计算各类均值
4. **更新类中心**：以各类均值作为新的类中心点
5. **重复迭代**直至达到最大迭代次数或收敛标准
6. 输出聚类结果

## 方法对比总结

| 对比维度 | 层次聚类 | 快速聚类（K-Means） |
|---------|---------|-------------------|
| **适用样本量** | 中小样本 | 大样本 |
| **内存占用** | 较大 | 较少 |
| **聚类数目** | 事后根据树状图决定 | 事先由用户指定 $K$ |
| **距离基础** | 多种类间距离可选 | 基于欧氏距离 |
| **核心输出** | 树形图、冰柱图、聚结表 | 类成员表、最终类中心 |
| **迭代机制** | 无迭代，逐步合并 | 多次迭代更新类中心 |
