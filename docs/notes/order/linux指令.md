## 🐧 Linux 系统基本命令整理

### 1\. 文件和目录操作

| 命令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `ls` | 列出目录内容。常用选项：<br> `-l` (长格式显示，包含权限、所有者等)<br> `-a` (显示所有文件，包括隐藏文件/以 `.` 开头的文件)<br> `-h` (以人类可读格式显示文件大小) | `ls -lha /home` |
| `cd` | 切换目录。<br> `cd ..` (返回上一级目录)<br> `cd ~` (返回用户主目录)<br> `cd -` (返回上一个工作目录) | `cd /etc/nginx` |
| `pwd` | 显示当前工作目录的路径。 | `pwd` |
| `mkdir` | 创建新目录。<br> `-p` (递归创建目录，即使上层目录不存在) | `mkdir -p project/src` |
| `rmdir` | 删除空目录（不常用，通常使用 `rm -r`）。 | `rmdir empty_folder` |
| `touch` | 创建空文件或更新文件时间戳。 | `touch newfile.txt` |
| `cp` | 复制文件或目录。<br> `-r` (递归复制目录) | `cp file.txt new_location/`<br>`cp -r my_folder backup/` |
| `mv` | 移动或重命名文件/目录。 | `mv old_name.txt new_name.txt`<br>`mv file.txt /tmp/` |
| `rm` | 删除文件或目录。<br> `-f` (强制删除，无提示)<br> `-r` (递归删除目录及其内容) | `rm file_to_delete.txt`<br>`rm -rf unwanted_folder/` |

### 2\. 文件内容查看和编辑

| 命令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `cat` | 连接文件并打印到标准输出（常用于查看短文件）。 | `cat /etc/hosts` |
| `less` | 分页查看文件内容，可向上/下滚动。按 `q` 退出。 | `less big_log_file.log` |
| `head` | 查看文件的前几行（默认 10 行）。<br> `-n N` (查看前 N 行) | `head -n 5 file.txt` |
| `tail` | 查看文件的末尾几行（默认 10 行）。<br> `-f` (实时监控文件末尾的新增内容) | `tail -f /var/log/syslog` |
| `grep` | 在文件中搜索匹配指定模式的行。 | `grep 'error' /var/log/messages` |
| `vi` / `vim` | 强大的文本编辑器。 | `vim config.conf` |

### 3\. 系统信息和状态

| 命令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `df` | 报告文件系统的磁盘空间使用情况。<br> `-h` (人类可读格式) | `df -h` |
| `du` | 估算文件或目录的磁盘空间使用量。<br> `-sh` (总计并以人类可读格式显示) | `du -sh my_folder/` |
| `free` | 显示系统内存使用情况。<br> `-h` (人类可读格式) | `free -h` |
| `top` | 实时查看系统运行状态和进程信息。按 `q` 退出。 | `top` |
| `ps` | 显示当前运行的进程快照。<br> `ps aux` (显示所有用户的进程) | `ps aux` |
| `uname` | 打印系统信息（内核版本、操作系统名称等）。<br> `-a` (显示所有信息) | `uname -a` |

### 4\. 权限管理

| 命令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `chmod` | 改变文件或目录的权限。<br> **数字模式：** `u` (用户), `g` (组), `o` (其他), `a` (所有)<br> `r` (读=4), `w` (写=2), `x` (执行=1) | `chmod 755 script.sh`<br>（**7**=rwx, **5**=r-x, **5**=r-x） |
| `chown` | 改变文件或目录的所有者和所属组。 | `chown user:group file.txt` |
| `sudo` | 以超级用户权限执行命令。 | `sudo apt update` |

### 5\. 进程管理

| 命令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `kill` | 向进程发送信号（通常用于终止进程）。<br> `kill PID` (默认发送 SIGTERM 终止信号)<br> `kill -9 PID` (发送 SIGKILL 强制终止信号) | `kill 12345` |
| `bg` | 将暂停的进程放到后台运行。 | |
| `fg` | 将后台运行的进程调到前台。 | |
| `jobs` | 列出当前 shell 的后台作业。 | |

### 6\. 网络工具

| 命令 | 描述 | 示例 |
| :--- | :--- | :--- |
| `ping` | 测试网络连接和延迟。 | `ping google.com` |
| `ip addr` | 显示和配置网络接口信息（较新的发行版推荐使用）。 | `ip addr show` |
| `ifconfig` | 显示和配置网络接口信息（较老的发行版）。 | `ifconfig` |
| `ss` / `netstat` | 查看网络连接、路由表、接口统计等信息。 | `ss -tuln` (TCP/UDP/Listen/Numeric) |
| `ssh` | 安全地远程连接到另一台计算机。 | `ssh user@192.168.1.10` |

### 💡 实用小贴士

  * **获取帮助：** 使用 `man command_name` 来查看任何命令的完整手册页。例如：`man ls`。
  * **清屏：** 使用 `clear` 命令或按下 `Ctrl + L` 来清空终端屏幕。
  * **历史命令：** 使用 `history` 查看执行过的命令列表，使用上下箭头回顾命令。
  * **管道符：** 使用 `|` (管道符) 将一个命令的输出作为另一个命令的输入。例如：`ls -l | less`。
